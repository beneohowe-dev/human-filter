# Contributing to Human Filter

Thanks for considering a contribution.

Human Filter should stay small, honest and useful. The best changes make the core product better without turning it into a social media suite.

## Good contributions

- Better AI-tell detection
- Clearer rewrite prompts
- More useful tone-mode behavior
- Accessibility fixes
- Mobile layout improvements
- Local storage and privacy improvements
- Documentation that makes the project easier to understand

## Product guardrails

- Do not add forced login to the MVP.
- Do not add scheduling, engagement farming or automation features.
- Do not invent personal stories in generated output.
- Do not add dark patterns or fake AI detector scores.
- Do not make image generation the main product.

## Local setup

```bash
npm install
npm run dev
```

## Before opening a PR

- Run `npm run build`.
- Keep copy plain and direct.
- Keep UI changes calm, accessible and responsive.
- Explain what changed and why.
