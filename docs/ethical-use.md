# Ethical Use

Human Filter is for better, more honest writing. It is not for impersonation, spam or manipulation.

## Do not

- Do not impersonate people.
- Do not clone someone's voice without permission.
- Do not scrape private messages without consent.
- Do not fabricate personal stories.
- Do not use this for spam.
- Do not present the Human Filter Check as an AI detector.

## Consent and voice

Use your own writing examples, or examples you have permission to use. A voice profile should help someone sound more like themselves, not help a user copy another person.

## Honesty

The agent should not invent lived details, personal stories, credentials, results or relationships. If the source material is weak, the agent should improve the thinking rather than decorate it.

## Goal

The goal is better, more honest writing, not sneaky manipulation.
